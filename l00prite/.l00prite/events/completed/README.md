# Completed Events

Move (not copy) resolved events here with resolution notes after verification. A completed
event must add `resolved_at`, `resolving_agent`, `verification_summary`, `response_summary`,
`related_commit` (if available), and `outcome` (`resolved` | `rejected` | `blocked` |
`duplicate` | `unsafe`) — see `../README.md`.

Do not mark an event complete without verification or an explicit explanation for why verification could not be run.
